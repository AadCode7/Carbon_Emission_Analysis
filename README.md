# Exploring Global Carbon Emissions Data

This project analyzes global carbon dioxide (CO₂) emissions using real-world data from Our World in Data. The objective is to explore trends, compare countries, and understand the relationship between emissions, population, and economic indicators.

## Dataset

- Source: Our World in Data  
- File used: `owid-co2-data.csv`  
- Description:  
  The dataset contains country-level CO₂ emissions, GDP, population, and related metrics from 1950 to 2021.

## Objectives

### 1. CO₂ Emissions Over Time
- Analyze how emissions have grown from 1950 to 2021  
- Compare across income groups:
  - Low income  
  - Lower-middle income  
  - Upper-middle income  
  - High income  
- Evaluate both:
  - Total emissions  
  - Per capita emissions  


### 2. Top Emitting Countries
- Identify top 5 emitters for:
  - 1960  
  - 1990  
  - 2020  
- Compare:
  - Absolute emissions  
  - Population-normalized emissions  
- Analyze how rankings change over time  


### 3. GDP vs CO₂ Emissions
- Study relationship between:
  - GDP per capita  
  - CO₂ emissions per capita  
- Include only countries with population ≥ 5 million  
- Compare across:
  - 1978  
  - 1998  
  - 2018  
- Use bubble plots with population as marker size  


### 4. Wealth Distribution Over Time
- Analyze global GDP per capita distribution  
- Evaluate inequality trends from 1950 to 2020  
- Use distribution plots to observe shifts over decades
